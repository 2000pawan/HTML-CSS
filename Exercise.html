BDI(Bi-Directional Isolation) in HTML

The bdi element in HTML stands for "Bi-Directional Isolation." It is used to isolate a span of text that might be formatted in a different direction from other text outside it, such as text in a different language or with a different base directionality.

In the example below, usernames are shown along with the number of points in a contest. If the bdi element is not supported in the browser, the username of the Arabic user would confuse the text (the bidirectional algorithm would put the colon and the number "80" next to the word "User" rather than next to the word "points").

  • User hrefs: 40 points
  • User jdoe: 60 points
  • User ترحيب:80 points

Note: The bdi element is currently supported only in Firefox and Chrome.


BDO(Bi-Directional Override) in HTML

The bdo element in HTML stands for "Bi-Directional Override." It is used to explicitly override the bidirectional algorithm for a specific span of text. The bidirectional algorithm is responsible for handling the directionality of text in a mixed-script environment where text can be in left-to-right (LTR) or right-to-left (RTL) scripts.
